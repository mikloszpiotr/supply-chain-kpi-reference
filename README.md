#  Supply Chain KPI Reference

A structured, interactive reference of **50 supply chain KPIs** built as a GitHub Page — designed for analysts, planners, and supply chain professionals who want a fast, reliable lookup for metrics that matter.

---

## Why We Built This

Supply chain analytics work often starts with the same question: *"How do we actually calculate this KPI?"*

Definitions vary across teams, formulas get lost in spreadsheets, and implementation differs depending on whether you're working in Python, SQL, or Excel. This reference solves that by putting everything in one place — consistent, clear, and ready to use.

---

## Why It Matters

- ✅ Removes ambiguity around KPI definitions across teams
- ✅ Speeds up dashboard and reporting development
- ✅ Useful for onboarding analysts new to supply chain
- ✅ Bridges the gap between business meaning and technical implementation

---

## What's Inside

**50 KPIs across 5 categories:**

| Category | KPIs |
|---|---|
| Service Level & Customer Fulfillment | OTD, OTIF, Perfect Order Rate, Fill Rate, and more |
| Inventory | Turnover, DIO, Accuracy, Safety Stock, Forecast Bias, and more |
| Forecasting & Planning | MAPE, WAPE, MAE, RMSE, Plan Attainment, and more |
| Procurement & Supplier | Supplier OTIF, PPV, Defect Rate, Lead Time, and more |
| Warehouse & Logistics | Picking Accuracy, Labor Productivity, Cost to Serve, and more |

**Each KPI includes:**
- Definition
- Formula
- Why it matters
- Python implementation
- SQL implementation
- Excel implementation
